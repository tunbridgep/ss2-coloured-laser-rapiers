In some cases, some rapier colours may be undesirable. A classic example is
that you may want to get rid of black and white rapiers.

In order to do this, please edit user.cfg in your basic system shock 2 folder,
with numbers specifying a weight (0 disables a rapier colour). For example:

rapier_chance_blue 1
rapier_chance_red 1
rapier_chance_green 1
rapier_chance_purple 0
rapier_chance_orange 0
rapier_chance_yellow 0
rapier_chance_white 0.5
rapier_chance_black 0.5

rapier_chance_assassin_blue 1
rapier_chance_assassin_red 1
rapier_chance_assassin_green 1
rapier_chance_assassin_purple 0
rapier_chance_assassin_orange 0
rapier_chance_assassin_yellow 0
rapier_chance_assassin_white 0
rapier_chance_assassin_black 0
